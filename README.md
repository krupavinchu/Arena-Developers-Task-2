# Arena Developers Task 2

This repository contains the task submission for Arena Developers Task 2.

## Project structure

- `Arena-Developers-main 2/` - main project folder
  - `ecommerce-backend/` - Java Spring Boot backend for the e-commerce app
  - `app.js`, `index.html`, `style.css` - front-end assets at the workspace root

## How to use

1. Open the `Arena-Developers-main 2` folder in your IDE.
2. For the backend, navigate to `Arena-Developers-main 2/ecommerce-backend`.
3. Build and run using Maven or your Java IDE.

## Notes

- This repository was initialized locally and pushed to GitHub.
- Ensure Java and Maven are installed before running the backend.
