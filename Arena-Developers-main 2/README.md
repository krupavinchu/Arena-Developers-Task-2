# Nexus Agency — Business Website

A fully animated, multi-page business website built with HTML5, CSS3, and vanilla JavaScript.

## 📁 File Structure

```
nexus-website/
├── index.html    — Main HTML (all 5 pages)
├── style.css     — All styles, animations, responsive design
├── app.js        — All interactivity and animations
└── README.md     — This file
```

## 🚀 How to Run

Simply open `index.html` in any modern browser. No build tools or dependencies required — it loads Google Fonts via CDN.

## 📄 Pages

- **Home** — Hero, services preview, testimonials, pricing
- **Services** — Full services grid + 4-step process
- **About** — Company story, values, team grid
- **Work** — Filterable portfolio grid
- **Contact** — Validated contact form

## ✨ Features & Animations

- Dark / Light theme toggle
- Rainbow scroll progress bar
- Canvas particle system (80 floating particles)
- Cursor trail effect
- Ripple effect on all button clicks
- Floating bubbles in hero section
- Morphing gradient blob backgrounds
- Hero dashboard card with animated chart bars
- Floating metric pills
- Glowing card border pulse
- Spinning avatar with shape morph
- Marquee ticker strip
- Scroll-reveal on all sections
- Service cards: gradient reveal + shimmer + icon bounce on hover
- Portfolio: filter animation with staggered reveals + thumbnail zoom
- Team cards: tilt + overlay on hover
- Value list items: slide right on hover
- Process steps: lift + gradient reveal + number bounce on hover
- Contact items: slide + icon bounce on hover
- Footer social icons: spin + gradient on hover
- Gradient headline text continuously shifts colors
- Nav gradient underline slide on hover/active
- Nav shadow appears on scroll

## 🔧 Tech Stack

- HTML5 (semantic, accessible)
- CSS3 (custom properties, Grid, Flexbox, keyframe animations)
- Vanilla JavaScript ES6+
- Canvas API for particle system
- Google Fonts: Fraunces (serif) + Plus Jakarta Sans (sans)

## 📱 Responsive Breakpoints

- Desktop: 1200px max-width container
- Tablet: ≤ 900px — single column layouts
- Mobile: ≤ 640px — compact nav, stacked grids
